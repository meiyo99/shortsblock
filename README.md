# ShortsBlock

A Chrome extension to help you take control of your YouTube experience by blocking distracting content and reducing clickbait appeal.

## Features

- **Block YouTube Shorts**: Hide Shorts from your YouTube feed
- **Homepage Redirect**: Redirect YouTube homepage to your subscriptions
- **Hide Metrics**: Remove view counts, subscriber counts, and likes to reduce FOMO
- **Grayscale Mode**: Desaturate YouTube UI to make thumbnails less attention-grabbing
- **Comment Redaction**: Hide distracting YouTube comments
- **Global Toggle**: Easily enable/disable the extension
