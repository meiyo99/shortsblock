# ShortsBlock

Hi, I’m Mayuresh. Did you know that you have ~~rights~~ the right to watch YouTube without getting lobotomized? The ~~Constitution~~ Chrome Web Store says you do.

This is a little extension that hides Shorts from your feed and, optionally, turns the entire site gray so it stops acting like a casino.

## FAQ

**What is ShortsBlock?**

Despite what the name would have you believe, it is an extension that blocks YouTube Shorts.

**Does it block anything else?**

Yes. It automatically redirects the homepage to your subscriptions feed, so you don't get ambushed by the algorithm the second you open the site. It also hides view counts, subscriber counts, and likes. Because why do you care how many people watched a video anyway? Just watch the video.

**What about the comments?**

It hides those too. You're welcome.

**Can it make YouTube look... less appealing?**

There's a Grayscale Mode that desaturates the UI, turning those neon, open-mouthed thumbnails into sad, gray images that your dopamine receptors won't care about.

**Why did you make this?**

A lot of free time and a rapidly declining attention span. 

**Does it actually work?**

It totally works. If it doesn't work for you, try refreshing the page harder?

**Can I contribute?**

I guess.

**I actually want to watch just one Short right now, can I turn it off?**

Yes, there’s a global toggle to easily disable the extension. But this energy isn't encouraged.
