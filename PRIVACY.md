# ShortsBlock Privacy Policy

**Last Updated:** March 19, 2026

## Data Collection

ShortsBlock does NOT collect, transmit, or share any personal data.

## Data Storage

The extension stores your preferences locally using Chrome's `storage.sync` API:
- Extension enabled/disabled state
- Feature toggle states (Block Shorts, Redirect Homepage, Grayscale Mode, Hide Metrics, Redact Comments)

This data:
- Stays on your device and your Google account (via Chrome Sync)
- Is never sent to external servers
- Is never shared with third parties
- Can be cleared by uninstalling the extension

## Permissions

- **`storage`**: Store your preference settings locally
- **`declarativeNetRequest`**: Block YouTube Shorts URLs at the network level
- **`webNavigation`**: Detect YouTube homepage visits for redirect feature
- **`scripting`**: Inject CSS and JavaScript to hide UI elements
- **Host permissions (`*.youtube.com`)**: Required to apply features on YouTube pages

We will update this policy if our data practices change. Check the "Last Updated" date above.
